# Ensemble Metrics

## Installation
* from github:
    ```sh
    pip install git+https://github.com/hitachi-nlp/ensemble-metrics.git@master
    ```
* or from local: ( **recommended** for traceability)
    ```sh
    git clone https://github.com/hitachi-nlp/ensemble-metrics.git
    cd ensemble-metrics
    pip install -e .
    ```
